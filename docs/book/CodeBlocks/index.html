کد بلاک‌ها در Markdown

کد بلاک‌ها (Code Blocks) برای نمایش قطعه‌های کد به صورت قالب‌بندی شده و با حفظ فرمت اولیه آن استفاده می‌شوند. Markdown دو روش اصلی برای ایجاد کد بلاک‌ها ارائه می‌دهد: کد بلاک‌های درون‌خطی (Inline Code) و کد بلاک‌های چندخطی (Multiline Code Blocks)

کد بلاک‌های درون‌خطی (Inline Code)

کد بلاک‌های درون‌خطی برای نمایش قطعه‌های کوتاه کد در داخل یک خط متن استفاده می‌شوند. برای این کار، کافی است کد مورد نظر خود را بین دو علامت بک‌تیک (`) قرار دهید.

مثال:

نمونه کد 

    
      برای اجرای دستور `print("Hello, World!")` در پایتون، از این دستور استفاده کنید.

خروجی

برای اجرای دستور print("Hello, World!") در پایتون، از این دستور استفاده کنید

کد بلاک‌های چندخطی (Multiline Code Blocks)

کد بلاک‌های چندخطی برای نمایش قطعه‌های کد طولانی و چندخطی استفاده می‌شوند. برای ایجاد این نوع کد بلاک، می‌توانید از سه علامت بک‌تیک (```) یا تورفتگی چهار فاصله‌ای استفاده کنید.

نمونه کد 

    
      ```python 
 def greet(name):
    print(f"Hello, {name}!")

greet("Alice")

```

خروجی 

          
            :def greet(name)
            
print(f"Hello, {name}!")


greet("Alice")

برجسته‌سازی نحوی (Syntax Highlighting)

بسیاری از ویرایشگرهای Markdown و سیستم‌های مدیریت محتوا از برجسته‌سازی نحوی برای کد بلاک‌ها پشتیبانی می‌کنند. برای فعال کردن این قابلیت، زبان برنامه‌نویسی را پس از سه علامت بک‌تیک مشخص کنید.

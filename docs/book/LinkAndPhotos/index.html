کار با لینک ها و تصاویر در Markdown

لینک‌های خارجی برای ارجاع به منابع یا صفحات وب خارج از سند فعلی استفاده می‌شوند. برای ایجاد یک لینک خارجی، از ساختار زیر استفاده کنید: 

    
      [متن لینک](آدرس لینک)

متن لینک: متنی که داخل براکت‌ها [ ] قرار می‌گیرد، متنی است که به عنوان لینک نمایش داده می‌شود.

آدرس لینک: URL یا آدرس وب‌سایتی که می‌خواهید به آن ارجاع دهید، داخل پرانتز ( ) قرار می‌گیرد.

مثلا اگر بخواهیم یک لینک به سایت گوگل داشته باشیم به صورت زیر عمل میکنیم

نمونه کد 

    
      [وبسایت گوگل](https://www.google.com)

لینک‌های داخلی

لینک‌های داخلی برای ارجاع به بخش‌های مختلف در همان سند استفاده می‌شوند. برای این کار، ابتدا باید به بخش مورد نظر یک شناسه (anchor) اختصاص دهید و سپس از آن شناسه برای ایجاد لینک داخلی استفاده کنید.

1. ایجاد شناسه (Anchor)

شناسه‌ها معمولاً با استفاده از عناوین (Headers) ایجاد می‌شوند. به عنوان مثال، فرض کنید یک عنوان به صورت زیر دارید: 

    
      ## بخش مورد نظر

Markdown به طور خودکار برای این عنوان یک شناسه ایجاد می‌کند که معمولاً با خط تیره (dash) بین کلمات نمایش داده می‌شود. در این مثال، شناسه بخش-مورد-نظر خواهد بود.

2. ایجاد لینک داخلی

برای ایجاد یک لینک داخلی به این بخش، از ساختار زیر استفاده کنید:

    
      [متن لینک](#شناسه)

استفاده از تصاویر در Markdown

افزودن تصاویر به مستندات Markdown، زیبایی و خوانایی آنها را افزایش می‌دهد و اطلاعات بصری مفیدی ارائه می‌دهد. برای درج تصاویر در Markdown، از ساختار ساده‌ای استفاده می‌شود. 

    
      ![متن جایگزین](آدرس تصویر)

متن جایگزین (alt text): متنی که در صورت عدم بارگذاری تصویر نمایش داده می‌شود. این متن برای دسترسی‌پذیری و بهینه‌سازی موتورهای جستجو نیز مفید است. آدرس تصویر: URL یا مسیر محلی فایل تصویری که می‌خواهید نمایش داده شود.

برای مثال میتونیم یک عکس از Lorem Picsum رو اینجوری نشون بدیم
